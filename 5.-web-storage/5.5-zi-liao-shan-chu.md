# 5.5 資料刪除

## localStorage 資料刪除

可透過以下方式來刪除 localStorage 裡的資料：

```javascript
// 刪除單一項目
localStorage.removeItem("colorSetting2");

// 清空
localStorage.clear();
```



## sessionStorage 資料刪除

可透過以下方式來刪除 sessionStorage 裡的資料：

```javascript
// 刪除單一項目
sessionStorage.removeItem("colorSetting_from_session2");

// 清空
sessionStorage.clear();
```
